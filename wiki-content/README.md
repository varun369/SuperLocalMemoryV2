## What's Updated for v2.1.0

### New Features Documented

1. **MCP Integration**
   - 11+ IDE support (Claude Desktop, Cursor, Windsurf, Continue.dev, ChatGPT, Perplexity, Zed, OpenCode, Antigravity, Cody, Aider)
   - Auto-configuration during installation
   - Manual setup guides for each IDE
   - 6 MCP tools, 4 resources, 2 prompts

2. **Universal Skills**
   - 6 slash-commands (slm-remember, slm-recall, slm-list-recent, slm-status, slm-build-graph, slm-switch-profile)
   - Compatible with Claude Code, Continue.dev, Cody
   - Metadata-first design
   - Complete usage documentation

3. **Universal Architecture**
   - Expanded from 4 layers to 7 layers
   - Added Universal Access layer
   - Added MCP Integration layer
   - Added Skills layer
   - Single database for all access methods
